## FAQ

Q: What platform is UI Util designed to work on?
A: UI Util is designed to work on macOS and Chrome. Features may not work on Windows or Linux

Q: Can I use it on mobile?
A: UI Util does not work on most mobile device as console is unavaliable.

Q: How do I hide the options?
A: You can hide options either with right shift or F9.

Q: How do I change settings?
A: You can modify individual settings of features by right clicking the element.

Q: Why does UI Util not work on some websites?
A; Some websites enforce strict policies not allowing connections with our servers. We will fix this shortly by adding a Chrome Extention.
